# Ethics

## Intended Use

This repository is intended for self-consented personalized avatar research and engineering.

## Boundaries

- do not use the project as a celebrity impersonation system
- do not publish deceptive synthetic media without disclosure
- keep generated samples clearly labeled as synthetic
- treat public-figure footage, if used at all, as limited internal stress-test data

## Safety Practices

- prefer self-recorded data with clear consent
- document limitations and failure modes
- consider watermarking or visible disclosure for shared samples
